**모티브**: 던그리드, 스컬 더 히어로 슬레이어
**엔진**: Unity 6.3 / 1인 개발 / 2D 횡스크롤 로그라이크 액션

## 1. 구현해야 할 것

### 플레이어
- 무빙 (WASD / 방향키)
- 공격 (마우스 포인터 방향 기반 — 근거리 / 원거리 둘 다)
- 죽음 처리
- 무기 교체 시스템
- 세이브 / 로드 시스템

### 적
- 무빙
- 공격 (근거리 / 원거리)
- 타겟 추격 AI
- 죽음 처리
- 보스 몬스터 1체
- 종류는 진행 상황 보고 결정

### 맵 생성
- 방 프리팹 제작
- 몹 스폰 매니저 제작
- 생성 방식 미정 — 사전 제작 방 랜덤 배치 권장

### 아이템
- 생성 / 교체 시스템
- 스트링 테이블 사용
- 상황 보면서 최대한 많이 추가

### 사운드 / 연출
- BGM
- 효과음
- 파티클

### 옵션
- 일시정지 토글

## 2. 추가로 고려할 것 (누락되기 쉬운 부분)

### 시스템
- HP / 무적시간 / 피격 판정
- 히트박스 / 허트박스 분리
- 타격감 (카메라 셰이크, 히트스톱, 넉백)
- 오브젝트 풀링 (총알, 파티클, 적)
- 게임오버 / 리스타트 / 다음 방 진행 흐름

### UI
- HP바, 무기 슬롯, 아이템 보유 표시
- 타이틀 / 게임오버 / 클리어 화면
- 일시정지 메뉴 (옵션 진입 포함)
- 보상 / 아이템 획득 화면

### 입력
- Unity 6 신 Input System 사용 — 키 리바인딩 / 마우스+키보드 동시 처리에 유리

### 데이터
- ScriptableObject로 무기 / 아이템 / 적 데이터 분리
- 세이브: JSON + `Application.persistentDataPath` (PlayerPrefs 비추)
- 스트링 테이블은 Localization Package 사용

### 로그라이크 요소
- 영구죽음 / 런 단위 진행
- 방 클리어 보상 (아이템 선택)
- 메타 진행 (런 사이 영구 강화) — 여유 있으면

### 에셋
- 아트 / 사운드 에셋스토어 / itch.io 무료 활용
- 직접 제작은 시간 잡아먹음

### 버전 관리
- Git + GitHub Private
- `.gitignore`에 Library, Temp, Logs, Build, obj 추가
- Unity Smart Merge 설정

### 빌드
- 매주 1회 빌드 확인 — 마지막에 깨지면 답 없음
